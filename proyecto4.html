Consulta SQL para realizar la extracción, transformación y carga (ETL)


Extracción y Consolidación de Datos

Este script automatiza la extracción de información desde boletas de honorarios en formato PDF, consolidando los datos en un archivo Excel y CSV. Utiliza Python para leer los documentos, identificar los campos clave y estructurar la información en un formato tabular.

Tecnologías Utilizadas

  • Lenguaje: Python
  • Bibliotecas:
    • os: Manejo de archivos y carpetas.
    • pandas: Organización y exportación de datos en Excel y CSV.
    • PyPDF2: Lectura y extracción de texto desde documentos PDF.

Flujo del Script

1. Recorrido de la Carpeta

Se explora una carpeta en busca de archivos PDF.

2. Extracción de Información Clave

Se extraen datos de cada boleta utilizando palabras clave, incluyendo:

  • Nombre del contribuyente
  • Institución o receptor
  • RUT del receptor y contribuyente
  • Número de boleta
  • Fecha de emisión
  • Detalle de atención profesional
  • Monto total de honorarios
  • Impuesto retenido
  • Total final después de impuestos
3. Consolidación de Datos

Se almacena la información de todas las boletas en un DataFrame de Pandas.

4. Exportación a Formatos Estandarizados

Se generan reportes en Excel (.xlsx) y CSV (.csv), permitiendo su análisis en herramientas como Microsoft Excel o Google Sheets.

¿Por qué es útil?

  • Permite automatizar la gestión de documentos financieros, reduciendo el trabajo manual.
  • Estandariza la información, facilitando la auditoría y el análisis de datos.
  • Ahorra tiempo y minimiza errores en la consolidación de boletas de honorarios.

Este script demuestra cómo Python y Pandas pueden ser utilizados para extraer, transformar y estructurar datos desde archivos PDF, un proceso clave en la digitalización y automatización de la gestión documental. 🚀
